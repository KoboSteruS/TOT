# 💬 Чат-бот ООО «ТОТ»

## Общее описание

Чат-бот — это **интеллектуальный помощник**, интегрированный в сайт, который отвечает на вопросы посетителей автоматически на основе базы знаний.

**Источник данных**: `static/faq.json` (~53KB, 486 строк)  
**Движок**: Vanilla JavaScript (без внешних библиотек)  
**Интеграция**: Fixed-виджет в правом нижнем углу

---

## 🎯 Основные возможности

### 1. Автоматические ответы
- Поиск по ключевым словам
- Мгновенный ответ (без задержки)
- Контекстно-зависимые ответы

### 2. Популярные вопросы
- 8 быстрых кнопок с частыми вопросами
- Одним кликом — получить ответ

### 3. История переписки
- Сохранение всех сообщений в сессии
- Прокрутка истории
- Визуальное разделение: вопросы пользователя / ответы бота

### 4. Умный fallback
- Если бот не нашёл ответ — предлагает связаться с менеджером
- Все контакты (телефон, email, VK)

## 🧩 Структура FAQ

### Формат интента

```json
{
  "intent": "unique_intent_name",
  "keywords": ["ключ1", "ключ2", "синоним1"],
  "response": "Текст ответа бота",
  "category": "general|services|tariffs",
  "price": "от 10 000 ₽" (опционально)
}
```

### Поля

| Поле | Тип | Описание | Обязательное |
|------|-----|----------|--------------|
| `intent` | string | Уникальный ID вопроса | ✅ |
| `keywords` | array | Ключевые слова для поиска | ✅ |
| `response` | string | Ответ бота | ✅ |
| `category` | string | Категория (general/services/tariffs) | ✅ |
| `price` | string | Цена услуги (для отображения) | ❌ |

## 🔍 Алгоритм поиска ответов

### Шаг 1: Нормализация вопроса
```javascript
const normalizedQuery = userMessage.toLowerCase().trim();
```

### Шаг 2: Поиск по keywords
```javascript
for (let item of faqData.faq) {
  const matches = item.keywords.some(keyword => 
    normalizedQuery.includes(keyword.toLowerCase())
  );
  
  if (matches) {
    return item.response;
  }
}
```

### Шаг 3: Fallback (если не найдено)
```javascript
return `К сожалению, я не нашёл ответ на ваш вопрос. 
Свяжитесь с нами: 
📞 +7 (964) 319-46-46
✉️ info@tot-ptz.ru
💬 VK: vk.com/oootot`;
```

## 💾 Технические детали

### Загрузка FAQ

```javascript
let faqData = null;

async function loadFaq() {
  try {
    const response = await fetch('/static/faq.json');
    faqData = await response.json();
  } catch (error) {
    console.error('Ошибка загрузки FAQ:', error);
  }
}

// Загрузка при инициализации
document.addEventListener('DOMContentLoaded', loadFaq);
```

### Отправка сообщения

```javascript
function sendMessage(messageText) {
  if (!messageText.trim()) return;
  
  // Добавляем сообщение пользователя
  addMessage(messageText, 'user');
  
  // Поиск ответа
  const response = findAnswer(messageText);
  
  // Добавляем ответ бота (с задержкой для реалистичности)
  setTimeout(() => {
    addMessage(response, 'bot');
  }, 500);
}
```

### Добавление сообщения в чат

```javascript
function addMessage(text, sender) {
  const messageDiv = document.createElement('div');
  messageDiv.className = `chat-message ${sender}`;
  messageDiv.textContent = text;
  
  messagesContainer.appendChild(messageDiv);
  messagesContainer.scrollTop = messagesContainer.scrollHeight;
}
```

## 📝 Best Practices

### 1. Написание keywords

✅ **Хорошо**:
```json
"keywords": ["бухгалтерия", "ведение учета", "бухучет", "учет", "сопровождение"]
```

❌ **Плохо**:
```json
"keywords": ["бухгалтерское сопровождение"]  // Слишком длинное, не сработает при "бухучет"
```

### 2. Написание responses

✅ **Хорошо**:
```
Бухгалтерское сопровождение включает:
- Первичную документацию
- Учёт операций
- Отчётность в ФНС/ПФР/ФСС

Стоимость: от 15 000₽/мес.
Звоните: +7 (964) 319-46-46
```

❌ **Плохо**:
```
Мы делаем бухгалтерию.  // Слишком кратко, нет деталей
```

### 3. Категории

Используйте правильные категории для группировки:
- **general** — общие вопросы, не связанные с конкретными услугами
- **services** — описание услуг, цены, сроки
- **tariffs** — пакеты услуг, тарифные планы

## 🐛 Troubleshooting

### Бот не отвечает

**Причина**: FAQ не загрузился

**Решение**:
1. Откройте консоль браузера (F12)
2. Проверьте ошибки в сети (Network tab)
3. Убедитесь, что `faq.json` доступен по адресу `/static/faq.json`

---

### Бот отвечает неправильно

**Причина**: Keywords не совпадают с вопросом

**Решение**:
1. Добавьте больше синонимов в `keywords`
2. Используйте более общие ключевые слова
3. Проверьте регистр (всё приводится к lowercase)

---

### Некорректное отображение ответа

**Причина**: Спецсимволы или длинный текст

**Решение**:
1. Используйте escape-последовательности для `"` → `\"`
2. Разбивайте длинные ответы на абзацы
3. Используйте `\n` для переносов строк (если поддерживается)
