# 🎨 Секции сайта ООО «ТОТ»

## Обзор структуры сайта

Сайт состоит из **13 основных секций**, расположенных вертикально (Single Page Application):

```
┌─────────────────────────────┐
│  1. Hero (Главный экран)    │ #hero
├─────────────────────────────┤
│  2. О компании              │ #about
├─────────────────────────────┤
│  3. Услуги                  │ #services
├─────────────────────────────┤
│  4. CTA Консультация        │ #cta-modern
├─────────────────────────────┤
│  5. Как мы работаем         │ #process
├─────────────────────────────┤
│  6. Партнёры (Альфа-Банк)   │ #partner
├─────────────────────────────┤
│  7. Масштабы бизнеса        │ #business-scales
├─────────────────────────────┤
│  8. Тарифы                  │ #tariffs
├─────────────────────────────┤
│  9. Форма заявки            │ #final-cta
├─────────────────────────────┤
│ 10. Как нас найти (Карта)   │ #map
├─────────────────────────────┤
│ 11. Footer (Подвал)         │
├─────────────────────────────┤
│ 12. Чат-бот (Fixed)         │
└─────────────────────────────┘
│ 13. Модальные окна          │
```

  - Услуги (secondary)

- Полноэкранный блок
- Градиентный оверлей на фото
- Крупная типографика
- Две CTA-кнопки с разными стилями
- Адаптив: на мобилке текст уменьшается, кнопки в столбец

## 13. 🪟 Модальные окна

### 1. **Модалка "Все услуги"**
- Триггер: Кнопка в секции Services
- Содержит: Полный список всех услуг с ценами
- Источник данных: Отдельные интенты в `faq.json` (accounting_service, tax_accounting, и т.д.)

### 2. **Модалка "Почему ООО «ТОТ»?"**
- Триггер: Кнопка в секции About
- Содержит: История названия компании (бог Тот)
- Источник данных: `content.json → about.modal_*`

### 3. **Модалка "Сертификаты"**
- Триггер: Кнопка в секции Partner
- Содержит: Галерея сертификатов (загружается через API)
- Источник данных: `certificates.json` (динамическая загрузка)

### 4. **Модалка "Увеличение сертификата"**
- Триггер: Клик по сертификату в галерее
- Содержит: Full-size изображение сертификата

### Общие особенности модальных окон
- Overlay с затемнением
- Анимация появления
- Закрытие по клику на overlay или кнопку
- Прокрутка внутри модалки
- Блокировка прокрутки body при открытой модалке

## Адаптивность

### Breakpoints
- **Desktop**: > 1024px
- **Tablet**: 768px - 1024px
- **Mobile**: < 768px
- **Small mobile**: < 480px

### Ключевые изменения на мобилке
- Grid-секции переходят в 1 колонку
- Размер шрифтов уменьшается
- Кнопки в столбец
- Навигация упрощается
- Изображения скрываются или уменьшаются
- Отступы уменьшаются
