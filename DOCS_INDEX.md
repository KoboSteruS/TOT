## 🔑 Ключевые концепции

### 1. **Разделение данных и представления**
Весь контент хранится в JSON-файлах, а не хардкодом в HTML. Это позволяет:
- Редактировать через админ-панель
- Легко добавлять переводы
- Версионировать контент отдельно от кода

### 2. **JWT-авторизация**
Безопасный доступ к админ-панели через токены:
- Токен в URL → проверка → доступ
- Все API защищены JWT Bearer токеном
- Секреты в `.env` файле

### 3. **RESTful API**
Все операции через стандартные HTTP методы:
- `GET` — получение данных
- `POST` — создание
- `PUT` — обновление
- `DELETE` — удаление

### 4. **Интеллектуальный чат-бот**
Поиск по ключевым словам в FAQ:
- Мгновенные ответы без задержки
- 100+ готовых ответов
- Популярные вопросы кнопками
- Fallback на контакты

### 5. **Адаптивный дизайн**
Mobile-first подход:
- Breakpoints: 1024px, 860px, 768px, 480px
- Grid → 1 колонка на мобилке
- Уменьшение шрифтов
- Скрытие/адаптация изображений
